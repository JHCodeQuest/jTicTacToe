# Tic Tac Toe

A feature-rich Tic Tac Toe game with AI opponent, multiple difficulty levels, and local PvP mode.

## Play Now

**Live Demo:** https://jhcodequest.github.io/jTicTacToe/

## Features

- **Game Modes:** Player vs AI or Player vs Player
- **Difficulty Levels:** Easy, Medium, Hard
- **AI Smart:** Wins, blocks forks, plays optimally on Hard
- **Stats Tracking:** Per-difficulty win/loss/draw statistics
- **Win Streaks:** Fun alerts at 3+ consecutive wins
- **Dark Mode:** Toggle between light and dark themes
- **Sound Effects:** Move, win, lose, and draw audio feedback
- **Persist:** All stats and settings saved to localStorage
- **Animations:** Pop-in X/O placement, last move highlight

## Tech Stack

- Plain JavaScript (no frameworks)
- CSS Grid for board layout
- Web Audio API for sound
- LocalStorage for persistence
